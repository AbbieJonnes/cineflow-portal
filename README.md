# CINEFLOW CREATOR PORTAL

## Project Description

CineFlow Creator Portal is a multimedia registration webpage designed for upcoming content creators.  
The platform allows users to register, upload profile pictures, select content genres, and interact with multimedia features such as promotional videos.

The project was built using HTML and CSS while applying HTML5 form validation techniques and organized form structures.

## Features

- User registration form
- HTML5 form validation
- Multimedia integration using video
- Profile picture upload
- Content genre selection
- Weekly upload range slider
- Responsive and clean user interface
- Social media footer icons

## Technologies Used

- HTML
- CSS
- Font Awesome Icons
